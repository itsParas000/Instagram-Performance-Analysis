# Instagram Performance Analysis  

This is an Instagram Performance Analysis dashboard designed for social media managers and content creators.
It analyzes post-level data to track overall account growth, engagement quality, and content effectiveness.  

● Cleaned and processed 500 dummy Instagram performance records  
● Built pivot tables and charts to extract insights from raw data  
● Created an interactive dashboard showing content performance, monthly follower growth trends,
and reach by content type
